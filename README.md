# Machine Learning and Scientific Computing 2026

This public repository contains the material for the Machine Learning and Scientific Computing course.

## Contents

- **`lecture_notes/`**: Includes the PDF files for the theoretical lectures, as well as the complete merged lecture notes.
- **`practical_sessions/`**: Contains the Jupyter notebooks for the practical sessions (PCs). The notebooks are provided without solutions for students to complete.

## Environment Setup

To run the practical sessions, you'll need a Python environment with the required libraries. We recommend using `venv` or `conda`. 

```bash
# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
